# Summariser

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.2.22. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.


## Tech Stack
- Bun
- Express
- Prisma v7
- MySQL

## Setup

```bash
- bun install
- cp .env.example .env
- bunx prisma generate
- bun run dev
